Python-Dbms
-------------

Welcome to the documentation for **python-dbms**, a Python driver for DbmsDB_.

Requirements
=============

- DbmsDB version 3.7+
- Python version 3.7+

Installation
============

.. code-block:: bash

    ~$ pip install python-dbms --upgrade
